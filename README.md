## What This Is

A personal AI news digest system that:
- Aggregates the day's AI news from curated sources
- Generates a coherent 10-15 minute audio briefing
- Keeps a text ledger with source links for reference
- Delivers via WhatsApp voice note (or other channels)

## Why

Because doomscrolling Twitter/X for AI news is exhausting. One daily briefing > scattered anxiety.
